# BCD-Single-Digit-Multiplier

A hardware implementation of a BCD Single Digit Multiplier using only adders and basic logic gates.

During the design of combinatorial circuits for addition and subtraction, a natural curiosity emerged in extending these principles to implement the other basic operations like multiplication and division in hardware. A key challenge quickly identified was the representation of the tens digit, since conventional BCD +6 correction logic is primarily suited only for outputs less than 20.

This limitation served as a motivation to explore other existing approaches to convert binary bit sequences to decimal numbers. By reviewing the relevant literature, it became evident that the problem was tractable within the given hardware constraints with a structured and focused design approach. This realization served as the primary motivation for undertaking the cicuit level implementation of a Binary Coded Decimal (BCD) single digit multiplier.
